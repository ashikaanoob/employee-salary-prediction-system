# Employee Salary Prediction System

## Overview

Employee Salary Prediction System is a Machine Learning project developed using Python, Pandas, Scikit-Learn, and Streamlit.

The application predicts employee salaries based on:

* Experience
* Age
* Education Level
* Skill Score

## Features

* Data preprocessing using Label Encoding
* Multiple Linear Regression Model
* Interactive Streamlit Web Application
* Salary Prediction based on employee attributes
* Model evaluation using R² Score

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Streamlit

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation
8. Deployment using Streamlit

## Model Performance

R² Score: 0.9979

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
